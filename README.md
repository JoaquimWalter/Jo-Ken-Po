🎯 Descrição do Projeto:

Aplicação que permite jogar o game pedra, papel, tesoura. Desenvolvida com HTML, CSS e JavaScript. A interface permite ao usuário escolher uma opção dentre as 3 fornecidas, automaticamente a máquina fará a dela, exibindo a pontuação na tela e a contagem de vitórias.

🛠 Tecnologias Utilizadas

HTML: Estrutura.
CSS: Estilização.
JavaScript: Linguagem de programação.

📦 Funcionalidades

Escolha do usuário:
Pode escolher entre pedra, papel ou tesoura.

Escolha da máquina:
Fará uma escolha entre as três opções com um evento aleatório.

Resultado do vencedor: 
O resultado é contabilizado na tela para o vencedor da rodada.

💡 Resumo do Funcionamento:

O usuário escolhe uma opção, ao clicar na opção desejada, automaticamente a máquina seleciona a opção dela.
É contabilizado o valor do vencedor na tela.
